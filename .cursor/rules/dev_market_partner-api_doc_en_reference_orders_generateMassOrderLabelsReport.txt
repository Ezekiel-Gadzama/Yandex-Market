Ready‑made labels-stickers for all boxes in several orders
The method is available for models:FBS,ExpressandDBS.

• inventory-and-order-processing —Order processing and inventory
• all-methods — Full account management
Starts generation PDF-a file with labels for transferred orders. Details about why they are needed and what they look like are describedin the Help of the Market for sellers.
To display the external order ID on the label, pass it in the methodPOST v2/campaigns/{campaignId}/orders/{orderId}/external-id.
You can find out the generation status and get a link to the finished file using a request.GET v2/reports/info/{reportId}.

⚙️ Limit:1,000 requests per hour
