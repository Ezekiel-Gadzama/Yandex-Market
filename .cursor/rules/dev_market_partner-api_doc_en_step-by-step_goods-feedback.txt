Product Reviews
Yandex Market API allows you to receive product reviews and comments on them, as well as add, edit, and delete store comments.

Get product reviews
Enable API notifications Yandex.Market will send you a request.POST notificationwhen there is a new review.
How to work with notifications
Get feedback on the seller's products by requestingPOST v2/businesses/{businessId}/goods-feedback. To return reviews of certain products, in the parameterofferIdssend their IDs.

offerIds The results are returned page by page, one page contains no more than 50 reviews.

Get feedback that needs to be answered
Get the feedback you need to respond to with a requestPOST v2/businesses/{businessId}/goods-feedback, where pass the valueNEED_REACTIONin the parameterreactionStatus.

NEED_REACTION

reactionStatus
StatusNEED_REACTIONIt is removed after you respond to the review or skip it. If the author changes the review, the statusNEED_REACTIONit will appear again.

NEED_REACTION

NEED_REACTION
To leave a comment, send the ID of the review to be replied to and the text of the response in the request.POST v2/businesses/{businessId}/goods-feedback/comments/update.
If you don't want to respond to the review, skip it. — pass the review ID in the request.POST v2/businesses/{businessId}/goods-feedback/skip-reaction.

Get product comments
Enable API notifications Yandex.Market will send you a request.POST notificationwhen a new comment appears.
How to work with notifications If you already know the review IDs, skip the step of getting them.

• Get feedback on the seller's products by requestingPOST v2/businesses/{businessId}/goods-feedback.
• Pass the review ID in the request.POST v2/businesses/{businessId}/goods-feedback/commentsto see the comments on this review.
The results are returned page by page, one page contains no more than 50 comments.

Send a response to a review
If you already know the review IDs, skip the step of getting them.

• Get feedback on the seller's products by requestingPOST v2/businesses/{businessId}/goods-feedback.
• Send the ID of the review to be responded to and the text of the response in the request.POST v2/businesses/{businessId}/goods-feedback/comments/update.

Send a reply to a parent's comment
If you already know the IDs of the reviews and comments on the review, skip the steps to get them.

• Get feedback on the seller's products by requestingPOST v2/businesses/{businessId}/goods-feedback.
• Pass the review ID in the request.POST v2/businesses/{businessId}/goods-feedback/commentsto see the comments on this review.
• Send the ID of the parent comment to respond to and the text of the response in the request.POST v2/businesses/{businessId}/goods-feedback/comments/update.

Edit your comment
Is it possible to change the comment?
ParametercanModifyin the requestPOST v2/businesses/{businessId}/goods-feedback/comments/updateindicates whether the seller can change the comments.

canModify If you already know the IDs of the reviews and comments on the review, skip the steps to get them.

• Get feedback on the seller's products by requestingPOST v2/businesses/{businessId}/goods-feedback.
• Pass the review ID in the request.POST v2/businesses/{businessId}/goods-feedback/commentsto find out the ID of the comment that needs to be changed.
• Send the ID of this comment and the new text in the request.POST v2/businesses/{businessId}/goods-feedback/comments/update.

Delete your comment
Can I delete a comment?
ParametercanModifyin the requestPOST v2/businesses/{businessId}/goods-feedback/comments/updateindicates whether the seller can delete comments.

canModify If you already know the IDs of the reviews and comments on the review, skip the steps to get them.

• Get feedback on the seller's products by requestingPOST v2/businesses/{businessId}/goods-feedback.
• Pass the review ID in the request.POST v2/businesses/{businessId}/goods-feedback/commentsto find out the ID of the comment to delete.
• Pass the ID of this comment in the request.POST v2/businesses/{businessId}/goods-feedback/comments/delete.
