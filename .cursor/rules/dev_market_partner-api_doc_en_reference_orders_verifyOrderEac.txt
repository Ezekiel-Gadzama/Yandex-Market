Sending the confirmation code
The method is available for theExpressmodel.

• inventory-and-order-processing —Order processing and inventory
• all-methods — Full account management
Sends the confirmation code to the Market to verify it. If the store is configured to work with confirmation codes:
In the parameterdelivery, embedded inorder, the parameter is returnedeacTypewith the typeEnum(type of confirmation code for order transfer) in the methods:

Enum

• POST v1/businesses/{businessId}/orders;

• MERCHANT_TO_COURIER(temporarily non-refundable) — the seller transmits the code to the courier to receive a non-purchase;
