Transmitting an external order ID
The method is available for models:FBS,ExpressandDBS.

• inventory-and-order-processing —Order processing and inventory
• all-methods — Full account management
Transmits the order ID in the store's system to the Market. It cannot be transmitted to:

• more than once;
• after the order status changesPROCESSINGwith a sub-statusREADY_TO_SHIP.

PROCESSING

READY_TO_SHIP
This identifier is reflected in the shipping documents: the acceptance certificate, the assembly sheet, and the labels.
Barcodes can only contain characters ASCII
If you use other characters in the external identifier, the label in the barcode will display the Market's order ID.

⚙️ Limit:10,000 requests per hour

Request
POST

https://api.partner.market.yandex.ru/v2/campaigns/{campaignId}/orders/{orderId}/external-id
