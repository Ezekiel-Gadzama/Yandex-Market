Getting available chats
The method is available forall models.

• communication —Customer communication
• all-methods — Full account management
• all-methods:read-only — View all data
Returns chats with customers. Enable API notifications Yandex.Market will send you a request.POST notificationwhen a new chat or message appears.
How to work with notifications Restriction for the parameterlimit

limit Do not pass a value greater than 20.

⚙️ Limit:10,000 requests per hour
