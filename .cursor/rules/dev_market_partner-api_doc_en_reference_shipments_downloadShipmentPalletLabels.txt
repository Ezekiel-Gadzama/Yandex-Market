Labels for trust acceptance
The method is available for theFBSmodel.

• inventory-and-order-processing —Order processing and inventory
• all-methods — Full account management
PDF-a file with labels for each package in the shipment for confidential acceptance. Read more about this type of acceptance inYandex.Market Help for sellers.
Print out several copies of each label: you need to paste at least 2 labels from different sides on one package.
The number of packages in the shipment is passed in the methodPUT v2/campaigns/{campaignId}/first-mile/shipments/{shipmentId}/pallets.

⚙️ Limit:200 requests per hour

Request
GET
