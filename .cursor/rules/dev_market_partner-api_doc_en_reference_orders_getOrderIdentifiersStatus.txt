Returns the verification statuses of the marking codes in the order.
An order that contains jewelry or products with mandatory labeling in the system«Честный ЗНАК», can be converted to the statusREADY_TO_SHIP, only when:

READY_TO_SHIP

• In the methodPUT v2/campaigns/{campaignId}/orders/{orderId}/boxesYou will send the following information to Yandex.Market:Winfor each piece of jewelry in the order;marking codes in the system «Честный ЗНАК» for all products in the order for which it is required.
In the methodPUT v2/campaigns/{campaignId}/orders/{orderId}/boxesYou will send the following information to Yandex.Market:

• Winfor each piece of jewelry in the order;
Winfor each piece of jewelry in the order;
• marking codes in the system «Честный ЗНАК» for all products in the order for which it is required.
marking codes in the system «Честный ЗНАК» for all products in the order for which it is required.
• All marking codes will be successfully verified.
All marking codes will be successfully verified.

What is GTIN?GTIN — This is a unique number assigned to a product in a single international database.GS1. This number generates an EAN, UPC, or ISBN barcode.How to make sure that the product is in the databaseYou can check the code onverification pageon the GS1 association's website.
