Changing the order delivery date
The method is available for theDBSmodel.

• inventory-and-order-processing —Order processing and inventory
• all-methods — Full account management
The method changes the order delivery date in the statusPROCESSINGorDELIVERY. For orders with other statuses, the delivery date cannot be changed.

PROCESSING

DELIVERY

⚙️ Limit:100,000 requests per hour

Request
PUT

https://api.partner.market.yandex.ru/v2/campaigns/{campaignId}/orders/{orderId}/delivery/date

https://api.partner.market.yandex.ru/v2/campaigns/{campaignId}/orders/{orderId}/delivery/date
