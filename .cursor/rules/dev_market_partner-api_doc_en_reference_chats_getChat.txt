Getting a chat by ID
The method is available forall models.

• communication —Customer communication
• all-methods — Full account management
• all-methods:read-only — View all data
Returns the chat by its ID. Enable API notifications Yandex.Market will send you a request.POST notificationwhen a new chat or message appears.
