Receiving a chat message
The method is available forall models.

• communication —Customer communication
• all-methods — Full account management
• all-methods:read-only — View all data
Returns a message by its ID. Enable API notifications Yandex.Market will send you a request.POST notificationwhen a new chat or message appears.
How to work with notifications

⚙️ Limit:1,000 requests per hour
