Information about the goods you have hidden
The method is available forall models.

• offers-and-cards-management —Manage products and cards
• offers-and-cards-management:read-only —View products and cards
• all-methods — Full account management
• all-methods:read-only — View all data
Retrieves the list of hidden items for the specified store.
The list will contain items that are hidden in any way. — through the API, using the YML feed, in the cabinet, and so on.

⚙️ Limit:10,000 requests per minute, no more than 500 products per request

GET
