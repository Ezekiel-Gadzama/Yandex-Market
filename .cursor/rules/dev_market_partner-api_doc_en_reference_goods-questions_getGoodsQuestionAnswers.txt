Getting answers to a question
The method is available forall models.

• communication —Customer communication
• all-methods — Full account management
• all-methods:read-only — View all data
Returns answers to a question about a product using the specified filters. The results are returned page by page, one page contains no more than 50 responses.

⚙️ Limit:10,000 requests per hour

Request
POST

https://api.partner.market.yandex.ru/v1/businesses/{businessId}/goods-questions/answers
