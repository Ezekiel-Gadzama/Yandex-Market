Returns the bid values for the specified products. You will not be able to get information on campaigns created in the cabinet.
The response returns only the values of the rates that you set through the request.PUT v2/businesses/{businessId}/bids.
A single request can contain a maximum of 1,500 products.

⚙️ Limit:1,000 requests per minute
