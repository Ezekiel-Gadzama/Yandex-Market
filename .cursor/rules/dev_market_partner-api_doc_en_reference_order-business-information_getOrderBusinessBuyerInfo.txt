The method is available forall models.

• all-methods — Full account management
• all-methods:read-only — View all data
Returns information about the customer by the order ID. How to get information about a buyer who is an individual
Use the requestGET v2/campaigns/{campaignId}/orders/{orderId}/buyer. You can receive the data only if the order is in the statusPROCESSING,DELIVERY,PICKUPorDELIVERED.
