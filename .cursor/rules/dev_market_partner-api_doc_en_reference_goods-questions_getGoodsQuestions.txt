Getting questions about the seller's products
The method is available forall models.

• communication —Customer communication
• all-methods — Full account management
• all-methods:read-only — View all data
Returns questions about the seller's products using the specified filters. The results are returned page by page, one page contains no more than 50 questions.

⚙️ Limit:10,000 requests per hour

Request
POST

https://api.partner.market.yandex.ru/v1/businesses/{businessId}/goods-questions
