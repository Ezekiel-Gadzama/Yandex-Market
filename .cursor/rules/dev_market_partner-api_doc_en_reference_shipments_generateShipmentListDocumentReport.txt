Getting the Assembly sheet
The method is available for theFBSmodel.

• inventory-and-order-processing —Order processing and inventory
• all-methods — Full account management
Starts generationthe assembly sheetfor shipment.
In order for the external order ID to be displayed on the assembly sheet, pass it in the methodPOST v2/campaigns/{campaignId}/orders/{orderId}/external-id.
You can find out the generation status and get a link to the finished document using a request.GET v2/reports/info/{reportId}.

⚙️ Limit:100 requests per hour
