Deprecated The method is available forall models.

• all-methods:read-only — View all data
Which method should I use instead of the outdated one? POST v2/campaigns/{campaignId}/offer-prices
Returns a list of prices that you have set for products in any way: for example, via the API or in a catalog file.
How is the total number of items calculated? According to the data for the last seven days (not including today), it cannot exceed 2 million.
The methods of setting prices are describedin the Help of the Market for sellers.

⚙️ Limit:150,000 items per minute

Request
GET

https://api.partner.market.yandex.ru/v2/campaigns/{campaignId}/offer-prices
