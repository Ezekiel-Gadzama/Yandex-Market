Shipment confirmation
The method is available for theFBSmodel.

• inventory-and-order-processing —Order processing and inventory
• all-methods — Full account management
Confirms the shipment of goods to a sorting center or an order acceptance point. The action is available only after the shipment has been generated.
The shipment schedule is configured separately for each warehouse in your merchant profile and is not available via the API.
You can check the shipment confirmation using the methodGET v2/campaigns/{campaignId}/shipments/{shipmentId}: among the available actionsavailableActionsThere must be an actionCONFIRM.
Before the confirmation time, the method returns the code400and the error "Cutoff time for shipments has not been reached yet".
