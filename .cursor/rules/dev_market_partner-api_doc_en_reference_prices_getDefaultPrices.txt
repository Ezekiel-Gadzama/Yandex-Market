View prices for specified products in all stores
The method is available for models:FBY, FBS, Express and DBS.

• pricing —Manage prices
• pricing:read-only —View prices
• all-methods — Full account management
• all-methods:read-only — View all data
Returns a list of prices that you have set for all stores using any method. For example, through an API or using an Excel template.
Read about the ways to set pricesin the Help of the Market for sellers.

⚙️ Limit:10,000 items per minute
