Ready‑made label-sticker for the box in the order
The method is available for models:FBS,ExpressandDBS.

• inventory-and-order-processing —Order processing and inventory
• all-methods — Full account management
Generates a label sticker for the box in the order and returns the label to PDF‑ the file.
To display the external order ID on the label, pass it in the methodPOST v2/campaigns/{campaignId}/orders/{orderId}/external-id.

⚙️ Limit:100,000 requests per hour

Request
GET
