View prices for specified products in a specific store
The method is available for models:FBY,FBS,ExpressandDBS.

• pricing —Manage prices
• pricing:read-only —View prices
• all-methods — Full account management
• all-methods:read-only — View all data
Returns a list of prices for the specified products in the store. The method is only for individual stores
Use this method only if the cabinet has unique prices in individual stores. To view the prices that are valid in all stores, usePOST v2/businesses/{businessId}/offer-mappings.

⚙️ Limit:150,000 items per minute

POST
